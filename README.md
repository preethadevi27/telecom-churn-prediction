**# Telecom Customer Churn Prediction using logistic regression and random forest**



This project predicts customer churn in the telecom sector using **Logistic Regression** and **Random Forest Classifier.**  

It helps telecom companies identify customers who are likely to leave, enabling them to take proactive steps to retain them.

The dataset used is \*\*\[churn-bigml-20.csv]\*\*, which contains information such as:

\- International plan  

\- Voice mail plan  

\- Number of customer service calls  

\- Total call durations and charges  

\- Churn (Target variable: `True` or `False`)
